# Global Conflict Monitor

Global Conflict Monitor is a real-time dashboard that tracks geopolitical incidents using live social media signals and visualizes them on an interactive map.

## Features

- Live Twitter monitoring
- Interactive map visualization
- Incident alert popups
- Auto-refresh news feed
- Real-time OSINT monitoring

## Technologies Used

- React
- TypeScript
- D3.js
- Node.js API
- Vercel deployment

## Monitored Sources

- ALERTX360
- MonitorX99800

## Purpose

This project demonstrates how open source intelligence (OSINT) signals from social media can be used to monitor global conflict activity in real time.

## Disclaimer

This project is for educational and research purposes only. Information comes from publicly available sources and may not always be accurate.
